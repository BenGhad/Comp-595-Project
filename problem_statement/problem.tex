\section*{Input}
The first line of input consists of two integers: $N$ and $M$.
($1 \leq N \leq 200 ~ 000$, $2 \leq M \leq 2^{31} - 1$)
The next $2N$ lines consist of pairs of: a line which consists of
a single integer $k_i$ ($2 \leq k_i \leq 200 ~ 000$),
followed by a line which consists of $k_i$ integers $c_{i,j}$
($-2^{31} \leq c_i_j \leq 2^{31} - 1$),
which are the coefficients of polynomial $f_i$, i.e.
$f_i(x) = c_i_1 + c_i_2 x + c_i_3 x^2 + \dots + c_i_{k_i} x^{k_i - 1}$.

You are guaranteed that the sum of all $k_i$ does not exceed $200 ~ 000$.

\section*{Output}
Print $n$ lines each containing the smallest non-negative integer $x_i$
for which $f_i(x_i) \equiv 0 \mod M$.
